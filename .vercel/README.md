# Vercel Configuration

This directory is automatically created by Vercel CLI when you deploy.

## Quick Deploy Commands

```bash
# First time setup
vercel login
vercel

# Production deployment
vercel --prod

# Link to existing project
vercel link
```

## Project Configuration

- **Framework**: Vite
- **Build Command**: `npm run build:client`
- **Output Directory**: `dist/spa`
- **Node.js Version**: 18+ (auto-detected)
- **Python Version**: 3.11 (for deepfake API)

See `VERCEL_DEPLOYMENT.md` in the root directory for complete deployment instructions.
